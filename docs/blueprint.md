# **App Name**: ACADEX

## Core Features:

- Secure Authentication & Role-Based Access: Robust user login with forced password change on first access, automatic role assignment based on ID prefix (e.g., DIR-, ENS-, ELV-), and comprehensive security measures like account lockout, login history, automatic logout, and an audit action journal for all user activities.
- Dynamic Role-Specific Dashboards: Personalized real-time dashboards presenting critical information tailored to each user role. Directors see overall school statistics; Teachers view class assignments and subject data; Students and Parents access their academic progress and important updates.
- Comprehensive School Data Management (Director Tool): An administrative tool allowing Directors to manage all school entities including students, teachers, classes, subjects, timetables, and academic coefficients. This feature incorporates an AI-assisted provisioning tool for efficient user account generation and configuration suggestions.
- Teacher Grade & Absence Input System: Empowers teachers to efficiently record, update, and manage grades and attendance for their assigned students and classes, with support for absence justification and dynamic recalculation of student averages and ranks.
- Student/Parent Academic & Payment Portal: An intuitive portal enabling students and parents to view detailed academic records (individual notes, class average, highest/lowest scores, rank, and merit mentions), personalized timetables, absence history, agenda items, and payment statuses (paid, partial, unpaid).
- Integrated Messaging Platform: A secure, internal communication system facilitating direct and targeted messages between specified user groups: Director to Teachers, School to Parents, and School to Students, ensuring efficient information exchange.
- Financial Payment Tracking & Receipt Generation: A system to log and confirm physical payments made at the school, track student account balances (total, paid, remaining), manage payment statuses, and automatically generate professional PDF receipts for all validated transactions.

## Style Guidelines:

- Color scheme: Dark theme, chosen for a premium, modern, and professional aesthetic, providing excellent contrast and a focused user experience.
- Primary interactive color: A rich, deep green (#1A6B4A), symbolizing growth, learning, and stability.
- Background color: A very dark, subtle green-grey (#0D1413), providing a sophisticated backdrop that harmonizes with the primary color while ensuring content visibility.
- Accent color: A vibrant lime-green (#A7E24C), selected for high visibility and to draw attention to interactive elements and important information, ensuring strong contrast against the dark palette.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, tech-savvy, and clean aesthetic, perfect for titles and short bursts of information.
- Body text font: 'Inter' (sans-serif), chosen for its exceptional legibility and neutral, professional appearance, suitable for extensive content like notes, messages, and reports.
- Icons: 'Tabler Icons' will be utilized across the application, providing a clean, consistent, and modern visual language that complements the overall design philosophy.
- Overall Layout: Features a fixed top bar and a left-aligned sidebar, framing a scrollable main content area. This architecture ensures persistent navigation and flexible content display, optimized for responsive viewing across mobile, tablet, and desktop devices.
- Login Page: Designed professionally with a full-screen background image of smiling students in a positive atmosphere, subtly darkened by a light overlay. The login card will feature a lightweight glassmorphism effect, subtle blur, and rounded corners for a sleek, modern appearance.
- UI Elements: Modern shadows, subtle glassmorphism effects, and uniformly rounded corners will be consistently applied throughout the interface to reinforce the premium and professional design.
- Animations: Soft and subtle animations will be integrated, especially on interactive elements and the login card, to provide a smooth, fluid, and engaging user experience without distracting from content.
